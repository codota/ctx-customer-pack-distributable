# Onboard: Testing Lab (Step 1)

Analyzes your repository and generates a test plan.

## Usage

```bash
ctx-onboard step-1 --repo-path /path/to/repo --json
```

## Output

Generates `test-plan.yaml` with 5-10 test cases covering:
- Architecture questions
- Incident response scenarios
- Code intelligence queries
- Dependency analysis
- Documentation retrieval
